## What is YouDo?

YouDo is Samantha's automation engine. Instead of repeating the same actions every day — forwarding emails, creating tasks, setting reminders — you define a rule once and YouDo runs it for you in the background, every time.

It's designed for the work you already know you'll do. YouDo just does it first.

---

## How it works

Every YouDo rule has three parts:

<Steps>
  <Step title="Trigger">
    The event that kicks off the automation — an incoming email, a time of day, a task going overdue.
  </Step>
  <Step title="Conditions (optional)">
    Filters that narrow when the rule fires — a specific sender, a keyword in the subject, a time window.
  </Step>
  <Step title="Action">
    What Samantha does automatically — creates a task, drafts a reply, saves a note, adds a calendar event.
  </Step>
</Steps>

---

## Triggers

<Tabs>
  <Tab title="Email">
    - New email from a specific person or domain
    - Email contains certain keywords
    - Email matches a label or priority level
    - You send an email with a specific label (e.g. "needs follow-up")
  </Tab>
  <Tab title="Time">
    - Daily, weekly, or monthly schedule
    - Specific day + time (e.g. every Monday at 9 AM)
    - Relative timing (e.g. 3 days after an email is sent)
  </Tab>
  <Tab title="Tasks">
    - A task becomes due or overdue
    - A task is marked complete
  </Tab>
</Tabs>

## Actions

| Action | What it does |
|---|---|
| **Create a task** | Adds a task with due date, priority, and context |
| **Draft an email** | Prepares a reply for your review before sending |
| **Send an email** | Sends automatically (use carefully) |
| **Add a calendar event** | Blocks time or creates a reminder |
| **Save to Notes** | Captures key details from an email or thread |
| **Post a summary** | Summarizes a thread into a note + creates a follow-up task |

## Control & safety

<Info>
  YouDo rules can run in **auto mode** (fires immediately) or **draft mode** (Samantha prepares the action and waits for your approval). Start with draft mode for new rules.
</Info>

- **Start narrow** — Use specific senders or keywords to avoid false triggers
- **Draft before auto** — Review actions before switching to fully automatic
- **Pause anytime** — Rules can be toggled off without deleting them
- **Edit or delete** — Modify any rule at any time from the YouDo settings panel

## Getting started

The fastest way to set up YouDo is to describe a workflow you already repeat. Samantha will suggest the right rule structure.

<Tip>
  Try saying: *"Whenever I get an email from my accountant, create a task to reply by end of day."*
</Tip>
