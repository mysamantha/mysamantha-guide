## What is YouDo?

YouDo is Samantha's automation engine. Instead of repeating the same actions every day (forwarding emails, creating tasks, setting reminders) you define a rule once and YouDo runs it for you in the background when the trigger happens.

It's designed for the work you already know you'll do. YouDo handles it automatically (or prepares it for approval).

## How it works

Every YouDo rule has three parts:

<Steps>
  <Step title="Trigger">
    The event that kicks off the automation: an incoming email, a time of day, a task going overdue.
  </Step>
  <Step title="Conditions (optional)">
    Filters that narrow when the rule fires: a specific sender, a keyword in the subject, a time window.
  </Step>
  <Step title="Action">
    What Samantha does automatically: creates a task, drafts a reply, saves a note, adds a calendar event.
  </Step>
</Steps>

<Note>
  Available triggers and actions depend on your connected apps and permissions.
</Note>

## Triggers

<Tabs>
  <Tab title="Email">
    - New email from a specific person or domain
    - Email contains certain keywords
    - Email matches a label/category/importance (if available)
    - You send an email with a specific label (e.g. "needs follow-up")
  </Tab>
  <Tab title="Time">
    - Daily, weekly, or monthly schedule
    - Specific day + time (e.g. every Monday at 9 AM)
    - Time offsets (where supported), e.g. create a follow-up task 3 days later
  </Tab>
  <Tab title="Tasks">
    - A task becomes due or overdue
    - A task is marked complete
  </Tab>
</Tabs>

## Actions

| Action                       | What it does                                                                                                 |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------ |
| **Create a task**            | Adds a task with due date, priority, and context                                                             |
| **Draft an email**           | Prepares a reply for your review before sending                                                              |
| **Send an email (optional)** | Can send automatically if enabled, otherwise drafts for approval                                             |
| **Add a calendar event**     | Blocks time or creates a reminder                                                                            |
| **Save to Notes**            | Captures key details from an email or thread                                                                 |
| **Summary (optional)**       | Can summarize an email/thread and save it to Notes, and optionally create a follow-up task (where supported) |

## Example rules

<AccordionGroup>
  <Accordion title="VIP email → task">
    **Trigger:** Email arrives from your boss or a key client\
    **Action:** Create task "Reply to [Name]" due today, with a link/reference to the email as context
  </Accordion>
  <Accordion title="Invoice workflow">
    **Trigger:** Email subject contains "invoice"\
    **Condition:** Sender is external\
    **Action:** Create task "Process invoice" due tomorrow, filed under Finance
  </Accordion>
  <Accordion title="Daily planning routine">
    **Trigger:** Every day at 8:30 AM\
    **Action:** Create task "Plan today (10 min)" and include your top tasks (if available)
  </Accordion>
  <Accordion title="Follow-up safety net">
    **Trigger:** You send an email tagged "follow up"\
    **Action:** Create task "Follow up on email" due in 3 days, including a link/reference to the thread
  </Accordion>
  <Accordion title="Weekly review">
    **Trigger:** Every Friday at 4 PM\
    **Action:** Summarize completed tasks into a weekly note + create "Plan next week" task
  </Accordion>
</AccordionGroup>

## Control & safety

<Info>
  YouDo rules can run in **auto mode** (fires immediately) or **draft mode** (Samantha prepares the action and waits for your approval). Start with draft mode for new rules.
</Info>

- **Start narrow:** Use specific senders or keywords to avoid false triggers
- **Draft before auto:** Review actions before switching to fully automatic
- **Pause anytime:** Rules can be toggled off without deleting them
- **Edit or delete:** Modify any rule at any time from the YouDo settings panel

## Getting started

The fastest way to set up YouDo is to describe a workflow you already repeat. Samantha will suggest the right rule structure.

<Tip>
  Try saying: _"Whenever I get an email from my accountant, create a task to reply by end of day."_
</Tip>
