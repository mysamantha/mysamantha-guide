To Do works offline.\
AI-powered actions require an internet connection.

## **Creating Tasks**

You can create tasks in multiple ways:

1. Type directly in the To Do section
2. Use voice to add a task.
3. Ask Samantha to create a task from chat.
4. Turn notes, emails, or conversations into tasks.

You don’t need to set everything up front. Details can be added or changed later.

## **Voice to Task**

**What it does**\
Creates tasks from spoken input.

**How to use**

1. Tap the microphone
2. Say your task naturally.
3. The task is created with the detected details.

Free users can record up to **60 seconds**.\
Paid users can record up to **60 minutes**.

## **Smart Date Detection**

**What it does**\
Automatically detects dates and times from natural language.

**How it works**

1. Say or type things like “tomorrow,” “next Monday,” or “at 5 PM.”
2. Due dates are set automatically.

You can edit dates anytime.

## **Lists and Sections**

**What it does**\
Organizes tasks into structured groups.

**How to use**

1. Create multiple lists (for example: Work, Personal, Learning)
2. Divide lists into sections (for example: Urgent, Backlog, Ideas)

If no list is selected, tasks go into a global list by default.

## **Subtasks**

Subtasks are smaller steps inside a task. Use them like a checklist.

**How to use**

1. Create the main task (the outcome)
2. Add subtasks (the steps)
3. Tick subtasks, then complete the main task

**Best practices**

- Keep 3–7 subtasks max
- Start each with a verb - Draft, Review, Send
- Order them if sequence matters

**Examples**

- Prepare weekly report → Pull data → Draft → Review → Send
- Grocery run → Milk → Eggs → Rice

## **Priorities**

**What it does**\
Helps you focus on what matters most.

**How to use**

1. Assign a priority to any task.
2. Tasks are reordered automatically based on priority.

## **Reminders**

**What it does**\
Notifies you when it’s time to act.

**How to use**

1. Set a reminder for any task.
2. Get notified to complete, snooze, or reschedule

## **Smart Scheduling**

**What it does**\
Let Samantha manage task timing through conversation.

**How to use**

1. Ask Samantha to move, reschedule, or adjust tasks
2. Make bulk changes in seconds.

**Examples**:

- “Move my weekly report to Friday.”
- “Reschedule all client tasks to next week.”

## **Filters and Search**

**What it does**\
Helps you find the right tasks quickly.

**You can**

- Search tasks by meaning
- Filter by priority or list

This is useful when you remember the task but not where it lives.

## **Progress Tracking**

**What it does**\
Shows task status over time.

**You can view**

- Completed tasks
- Overdue tasks
- Archived tasks

This helps you understand progress and workload.

## **Archiving Tasks**

**What it does**\
Store tasks you don’t want to delete.

**How to use**

1. Archive tasks you’re not working on now
2. Restore them anytime

Archived tasks stay out of your active view.

## **Using To Do with Samantha**

You can manage tasks directly with the assistant.

**Examples**

- “Add a task to submit the report tomorrow.”
- “Change this task to high priority.”
- “Move my meeting prep to next week.”
- “What are my top tasks today?”

Samantha only acts when you ask.

## **Offline Behavior**

When offline, you can:

- Create tasks
- Edit tasks
- Change priorities
- Mark tasks as done
- Reorder lists

Sync happens atomatically once you’re back online.

## **Privacy & Control**

- Tasks are encrypted
- Your data is never sold.
- Tasks are not used to train AI models.
- You can export or delete tasks at any time. [more](https://mysamantha.ai/privacy)
